# 🌱 GreenHub — Mon Jardin Intelligent

**Ton jardin et ta serre pilotés par l'IA — 100% gratuit, offline-first**

## 🚀 Features

- 🗺️ **Jardin Réel (Canvas)** — Rendu isométrique avec Canvas 2D
- 🌱 **Mini-serres** — Gestion de serres avec stades de croissance
- 🎨 **Sprites réalistes** — Tomates, légumes et herbes générés procéduralement
- 📅 **Planning auto** — Tâches générées automatiquement
- 🌙 **Lune + Météo** — Meilleurs jours de plantation
- 🧮 **Calcul récoltes** — Estimations auto
- 🤖 **Conseils IA** — Pollinations.ai gratuit
- 📸 **Photo timeline** — Timelapse de tes plants
- 🏆 **Gamification** — Niveaux, badges, streaks
- 🛒 **Liste courses** — Auto-générée

## 📦 Installation

```bash
git clone https://github.com/avalondrey/greenhub
cd greenhub
npm install
npm run dev
```

## 🌐 Déploiement

### Vercel
```bash
npm run deploy:vercel
```

### GitHub Pages
```bash
npm run deploy:gh-pages
```

## 📊 Base de données plantes

50+ plantes incluses :
- 🍅 Tomates (20 variétés)
- 🥒 Légumes (15 variétés)
- 🌿 Herbes (10 variétés)
- 🍓 Fruits (5 variétés)

## 🛠️ Tech Stack

- React 19 + Vite 7
- IndexedDB (offline-first)
- PWA (installable mobile)
- DnD Kit (drag & drop)
- Open-Meteo API (météo gratuite)
